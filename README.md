# Customer Shopping Behaviour Analysis

An end-to-end analysis of customer shopping behaviour, combining **Python (EDA)**, **SQL (business analysis)**, and **Power BI (interactive dashboard)**.

## 📌 Project Overview

This project explores a retail customer dataset (3,900 records, 18 attributes) to understand purchasing patterns, customer segments, and revenue drivers. The workflow follows three stages:

1. **Exploratory Data Analysis (Python)** — cleaned and prepared the raw data
2. **Business Question Analysis (SQL)** — answered targeted business questions with queries
3. **Interactive Dashboard (Power BI)** — visualized key insights for stakeholders

## 🗂️ Repository Structure

```

customer_behaviour.sql                         # Business question queries

customer_behaviour.pbix                        # Power BI interactive dashboard

customer_shopping_behavior.csv                 # Raw dataset 
requirements.txt


## 🧹 Part 1: Exploratory Data Analysis (Python)

Using `pandas`, the notebook:
- Inspects data shape, types, and summary statistics
- Handles missing values in `Review Rating` (filled by category-level median)
- Standardizes column names (lowercase, underscores)
- Engineers new features:
  - `age_group` — quartile-based bins (Young Adult, Adult, Middle-aged, Senior)
  - `purchase_frequency_days` — numeric mapping of purchase frequency labels
- Removes redundant columns (`promo_code_used`, which duplicated `discount_applied`)
- Loads the cleaned data into a MySQL database for SQL analysis

> **Note:** Database credentials are read from environment variables (`DB_USER`, `DB_PASSWORD`, etc.) rather than hardcoded, so you can safely run this notebook without exposing secrets. Create a `.env` file or export these variables before running the load step.

## ❓ Part 2: Business Questions (SQL)

`sql/customer_behaviour.sql` answers questions including:

- What is the total revenue generated by male vs. female customers?
- Which customers used a discount but still spent above the average purchase amount?
- What are the top 5 products by average review rating?
- How do average spend amounts compare between standard and express shipping?
- Do subscribed customers spend more than non-subscribers?
- Which 5 products have the highest percentage of discounted purchases?
- How do customers segment into New / Returning / Loyal based on purchase history?
- What are the top 3 most purchased products within each category?
- Are repeat buyers (5+ purchases) more likely to hold a subscription?
- How does revenue contribution vary by age group?

## 📊 Part 3: Interactive Dashboard (Power BI)

`dashboard/customer_behaviour.pbix` brings the analysis together into an interactive dashboard covering revenue trends, customer segments, and product performance.

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>

# Install dependencies
pip install -r requirements.txt

# Launch the notebook
jupyter notebook notebooks/customer_shopping_eda.ipynb
```

To load data into MySQL for the SQL analysis, set environment variables first:
```bash
export DB_USER=root
export DB_PASSWORD=your_password
export DB_HOST=localhost
export DB_PORT=3306
export DB_NAME=customer_behaviour
```
